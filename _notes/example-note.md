This is an example note showing the structure and features available.

## Formatting

You can use all standard **Markdown** formatting:

- Bold, *italic*, and `code`
- [External links](https://example.com)
- Internal links like [digital-gardening](/notes/digital-gardening)

### Code Blocks

```javascript
function hello() {
  console.log("Hello, garden!");
}
```

### Blockquotes

> This is a blockquote. Use it for quotes or callouts.

## Linking to Other Notes

Link to other notes using standard Markdown links with the `/notes/` path:

```markdown
[Note Title](/notes/note-slug)
```

The note will automatically show "Linked References" at the bottom with backlinks from other notes that reference it.

## Metadata

Each note can have front matter with:

- `title`: The note title
- `date`: Creation date
- `status`: Growth stage (seedling, budding, evergreen)
- `tags`: List of tags for categorization
- `last_modified_at`: Optional update date
